# dotfiles

## dependencies

- neovim
- zsh
- git
- tmux
- alacritty
- starship
- alacritty
- gitui
- joshuto
- zimfw

## get source

```shell
    cd ~
    git clone --recursive https://github.com/kabirz/.dotfiles
    cd .dotfiles
```

## install
